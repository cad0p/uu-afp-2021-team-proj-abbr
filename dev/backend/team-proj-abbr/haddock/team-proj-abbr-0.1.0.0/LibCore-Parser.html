Map a string to a list of Tokens. For example: >>> parseInput "@bob" Right [DoMap (Keyword {keyword = "bob", plural = False})] >>> parseInput "@fw's" Right [DoMap (Keyword {keyword = "fw", plural = True})] >>> parseInput "hello!" Right [NoToken "hello!"]
